# MoodToday - 공부를 위한 날씨 관련 사이드 프로젝트

# Tech Stack

- Architecture Pattern: MVVM + Clean Architecture
- CI/CD: Fastlane + Jenkins
- View: SnapKit + Canvas
- Testing: XCTest + Quick + Nimble
- Network: Alamofire + Moya
- Other: SwiftLint, SwiftDate, IBAnimatable, RxSwift
